# Shopping List

## How to publish

Create APK with:

```shell
dotnet publish -f:net8.0-android -c:Release /p:AndroidSdkDirectory=path_to_sdk
```

For example:

```
dotnet publish -f:net8.0-android -c:Release /p:AndroidSdkDirectory=C:\Users\kbgoe\AppData\Local\Android\sdk
```
